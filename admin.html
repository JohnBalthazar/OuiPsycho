💬 Commentaires — Firebase (Firestore, gratuit)

Gratuit · Firestore en mode Spark · Règles de sécurité intégrées · Ouvrir Firebase Console →

Setup (3 min) :
  1. Firebase Console → votre projet → Firestore Database → créer la base (mode production)
  2. Onglet Règles → copiez le contenu de data/firebase-rules.txt et publiez
  3. Paramètres du projet (⚙) → Project ID et Web API Key ci-dessous
  4. Authentication → Sign-in method → Email/Mot de passe → activer
    Puis Authentication → Users → Ajouter un utilisateur (votre email admin)
  5. Renseignez email + mot de passe admin ci-dessous (stockés localement uniquement)
Paramètres du projet → Project ID
Paramètres du projet → Web API Key — safe à publier
L'utilisateur créé dans Firebase Authentication
Permet à l'admin de lire/modérer tous les commentaires

📬 Formulaire de contact (messages stockés dans Firebase — aucun service tiers)

Les messages du formulaire de contact sont stockés dans votre Firestore (collection contacts) et accessibles dans Admin → Messages. Aucun service tiers requis — Firebase suffit.

Prérequis : Firebase configuré ci-dessus + règles Firestore mises à jour (ajouter la collection contacts depuis data/firebase-rules.txt).
Affiché en dessous du formulaire comme alternative directe
